# Evernutri Rações - Site Institucional

Site institucional da Evernutri Rações Ltda, fabricante de rações para nutrição animal localizada em Conceição do Castelo, ES.

## Tecnologias

- HTML5
- CSS3 (com variáveis CSS e grid)
- JavaScript vanilla
- GitHub Pages (deploy automático)

## Estrutura

- `index.html` - Página principal
- `styles.css` - Estilos responsivos
- `script.js` - Interatividade e animações
- `CNAME` - Configuração de domínio customizado
- `.github/workflows/deploy.yml` - CI/CD automático

## Deploy

O site é automaticamente deployado via GitHub Actions sempre que há push na branch `main`.

**Domínio:** evernutri.com.br
